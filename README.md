# Portfolio – Moritz Leonard Mack 
Portfolio von Moritz Mack – Full-Stack PHP/Laravel Entwickler mit Projekten und Lebenslauf.


Dieses Repository enthält den Quellcode für mein persönliches Portfolio.  
Die Seite präsentiert meine beruflichen Schwerpunkte, ausgewählte Projekte sowie meinen Werdegang als Full Stack Developer mit Fokus auf **PHP, Laravel, SQL, Webanwendungen und Automatisierung**.

## Inhalte
- Übersicht meiner wichtigsten Projekte (z. B. Yachtauslastungs-Tool, Törnfinder, KI-Chatbot, MichiApp)
- Beruflicher Hintergrund und technische Schwerpunkte
- Kontaktinformationen und Download meines Lebenslaufs
- Modernes, statisches Frontend (HTML, CSS, JS)

## Technologie-Stack
Dieses Portfolio ist vollständig statisch aufgebaut:
- **HTML5**
- **CSS3** (responsive, moderne UI-Komponenten)
- **JavaScript** (kleine Interaktionen, Mouse-Blob-Effect)
- Keine Frameworks, kein Build-Prozess

Dadurch kann es schnell auf **GitHub Pages** oder jedem beliebigen Static Host bereitgestellt werden.

## Deployment (GitHub Pages)
1. Repository forken oder klonen  
2. Dateien ins Repo legen:  
   `index.html`, `style.css`, `script.js`, `img/`, `Lebenslauf.pdf`  
3. In den Repository-Einstellungen unter **Pages** →  
   „Deploy from Branch“ → `main` auswählen  
4. Portfolio ist live unter:  
   `https://{dein-github-username}.github.io/{repo-name}/`

## Lizenz
Dieses Portfolio-Projekt ist ausschließlich für persönliche Präsentationszwecke gedacht und steht nicht zur Wiederverwendung in kommerziellen Projekten.
